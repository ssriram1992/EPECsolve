28 # Running
29 - Open `Makefile`.
30 - Enter the path of your armadillo-installation in the line defining the variable `ARMA`. Typically, the location would be like `/opt/armadillo-code`.
31 - Enter the path of your Gurobi-installation in the line defining the variable `GUR`. Typically, the location would be like `/opt/gurobi/gurobi801/<Your OS>`.
32 - Run `make compileEPEC` to compile.
33 - Run `make` to run the code.
