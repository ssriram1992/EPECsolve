Starting from combination, the methods builds the recursion to generate the subproblems associated with all the existing combinations of polyhedra. Then, it solves each subproblem, and if a solution is found, it terminates and stores the solution into the referenced EPEC object. excludeList contains the excluded combinations of polyhedra.
